# Single cell course 2025W
Scripts from the Single Cell course 2025 winter semester. \
RNA-seq data that was used: https://www.ncbi.nlm.nih.gov/sra/SRX5822305[accn]


## Content of repository
| **File**     | Description |
| -------- | -------     |
| star_align.sh | Aligning RNA-seq reads |
| fastqc.sh | Quality control of the raw reads |
| trim.sh | Removing adapters and low-quality bases |
| trim_align.sh | Mapping RNA-seq reads to the reference genome |
| featureCounts.sh | Check how many RNA-seq fragments are mapped |
| stringtie.sh | Use STAR-aligned RNA-seq reads to assemble and report transcript structures |
| filtered_featureCounts.sh | Check mapped RNA-seq fragments again using the filtered gtf file |
| emapper.sh | |
| cellranger.sh | |
